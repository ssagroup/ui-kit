# ImageItem

## Default Usage

ImageItem is a versatile component that combines an avatar image with text content:

```tsx
import { ImageItem } from '@ssa-ui-kit/core';

function BasicImageItem() {
  return (
    <ImageItem
      image="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAABgAAAAYCAYAAADgdz34..."
      avatarSize={24}>
      John Smith
    </ImageItem>
  );
}
```

## Features

### Different Avatar Sizes

Customize the avatar size to match your design requirements:

```tsx
function AvatarSizes() {
  const user = {
    name: 'Sarah Johnson',
    image:
      'https://images.unsplash.com/photo-1494790108755-2616b6fe29c0?w=150&h=150&fit=crop&crop=face',
  };

  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: '16px' }}>
      {/* Small avatar for compact lists */}
      <ImageItem image={user.image} avatarSize={20}>
        {user.name} (Small - 20px)
      </ImageItem>

      {/* Default medium size */}
      <ImageItem image={user.image} avatarSize={24}>
        {user.name} (Default - 24px)
      </ImageItem>

      {/* Large avatar for emphasis */}
      <ImageItem image={user.image} avatarSize={32}>
        {user.name} (Large - 32px)
      </ImageItem>

      {/* Extra large for profile cards */}
      <ImageItem image={user.image} avatarSize={48}>
        {user.name} (Extra Large - 48px)
      </ImageItem>
    </div>
  );
}
```

# Switch

## Default Usage

The Switch component provides a toggle control that allows users to switch between two states (on/off, enabled/disabled). It's commonly used for settings, preferences, and feature toggles with clear visual feedback:

```tsx
import { Switch, SwitchContextProvider } from '@ssa-ui-kit/core';

function BasicSwitch() {
  return (
    <SwitchContextProvider initialState={false}>
      <Switch label="Enable notifications" />
    </SwitchContextProvider>
  );
}
```

## Features

### Color Variants

The `color` prop controls the on-state background, hover/focus shadow, and the
off-state focus outline. Colors are sourced from `theme.palette`.

| Value | Palette key | On background | Off outline |
|-------|-------------|---------------|-------------|
| `primary` *(default)* | `palette.primary` | `primary.main` | `primary.light` |
| `success` | `palette.success` | `success.main` | `success.light` |
| `custom` | — | via `colors.on` | via `colors.offOutline` |

The off state (unchecked) always shows a neutral grey background regardless of
color variant. The disabled state always uses `greyFocused40` (same for all variants).

```tsx
// Default — primary (blue)
<SwitchContextProvider initialState={false}>
  <Switch label="Enable feature" />
</SwitchContextProvider>

// Success (green)
<SwitchContextProvider initialState={false}>
  <Switch label="Enable feature" color="success" />
</SwitchContextProvider>

// Custom escape hatch
<SwitchContextProvider initialState={false}>
  <Switch
    label="Custom toggle"
    color="custom"
    colors={{ on: '#ff0000', offOutline: '#ff9999' }}
  />
</SwitchContextProvider>
```

### Disabled State

Disabled switches show a muted grey background in both on and off positions
and cannot be toggled:

```tsx
<SwitchContextProvider initialState={true}>
  <Switch label="Locked feature" isDisabled />
</SwitchContextProvider>
```

### Basic Toggle Functionality

Simple on/off switch with context provider:

```tsx
import { useState } from 'react';

function BasicToggleExample() {
  const [isEnabled, setIsEnabled] = useState(false);

  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: '16px' }}>
      <SwitchContextProvider initialState={isEnabled}>
        <div style={{ display: 'flex', alignItems: 'center', gap: '12px' }}>
          <Switch label="Dark mode" />
          <span>Dark mode</span>
        </div>
      </SwitchContextProvider>
      <SwitchContextProvider initialState={true}>
        <div style={{ display: 'flex', alignItems: 'center', gap: '12px' }}>
          <Switch label="Auto-save" />
          <span>Auto-save enabled</span>
        </div>
      </SwitchContextProvider>
    </div>
  );
}
```

# Icons

Icon set for the design system. Use the `Icon` component with a `name` from the list below. All icons share the same API: `name`, `size`, `color`, and optional `tooltip`.

## Usage

```tsx
import { Icon } from '@ssa-ui-kit/core';

<Icon name="user" size={24} color="#2B2D31" />
<Icon name="edit" size={20} />
<Icon name="search" color="#55575A" />
```
